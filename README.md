# Nova Edit（新星剪辑）

> AI 驱动的下一代智能视频剪辑软件 — 原型验证阶段

Nova Edit 是一款面向专业创作者与自媒体人的全平台智能剪辑工具。通过自研光影渲染引擎与 AI 深度集成，重新定义视频创作工作流。

---

## 项目状态

当前为 **Alpha 原型阶段**，包含三个核心模块的概念验证代码：

| 模块 | 文件 | 说明 |
|------|------|------|
| AI 语义剪辑引擎 | `prototype/ai_semantic_editor.py` | 场景分析、废片检测、文案成片、情绪匹配 |
| 骨骼级变速引擎 | `prototype/bone_speed.py` | 光流变速、自适应音频保持、高帧率升格 |
| 智能抠像引擎 | `prototype/smart_matting.py` | GrabCut 自动抠像、发丝边缘优化、环境光适配 |

## 快速开始

### 环境要求

- Python 3.10+
- OpenCV 4.8+
- （可选）FFmpeg — 音频处理功能需要

### 安装依赖

```bash
cd prototype
pip install -r requirements.txt
```

### 运行演示

```bash
cd prototype
python demo.py              # 完整端到端演示
python demo.py --quick      # 快速演示（跳过耗时操作）
python demo.py --skip-audio # 跳过音频处理（无 FFmpeg 环境时）
```

### 单独模块测试

```bash
cd prototype
python ai_semantic_editor.py   # AI 语义剪辑引擎
python bone_speed.py           # 骨骼级变速引擎
python smart_matting.py        # 智能抠像引擎
```

## 核心能力总览

### AI 语义剪辑引擎

- **场景自动分析**：帧间直方图差异检测镜头切换，自动生成章节结构
- **废片检测**：拉普拉斯模糊检测 + 亮度突变 + 静音段检测，综合质量评分
- **文案成片**：根据脚本文本自动从视频池分配镜头，按段落比例计算时长
- **情绪匹配**：中文情绪关键词（伤感/热血/平静/紧张）映射剪辑风格参数

### 骨骼级变速引擎

- **光流法变速**：Farneback 稠密光流 + 双向运动补偿中间帧插值
- **自适应音频保持**：librosa 时间拉伸 + 音高修正，避免变速变调
- **高帧率升格**：30fps → 240fps 光流插帧

### 智能抠像引擎

- **自动抠像**：GrabCut 自动 ROI + 迭代分割，输出透明背景 PNG
- **发丝级优化**：形态学闭运算 + 高斯羽化 + 边缘引导滤波
- **环境光适配**：色温分析 → 前景色彩匹配 → Alpha 混合合成
- **绿幕抠像**：HSV 色度键快速抠像

## 整体架构

Nova Edit 采用六层分层架构：

```
应用层（剪辑/调色/特效/音频/字幕/合成工作台）
  └─ UI 框架层（声明式 UI 引擎 / 时间线渲染 / 预览视口）
      └─ 业务逻辑层（8 大核心系统 / 插件运行时 / 工作流引擎）
          └─ 核心服务层（素材管理 / 版本控制 / 渲染调度 / AI 推理）
              └─ 渲染引擎层（自研光影内核 / GPU 调度 / 编解码）
                  └─ 硬件抽象层（Vulkan / Metal / D3D12 / NPU）
```

关键技术选型：C++20 / Rust / 自研 NovaSL 着色语言 / ONNX Runtime / FFmpeg / EGUI
